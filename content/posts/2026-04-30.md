### 1. Advances in protein function prediction from the fifth CAFA challenge
- Original Title: Advances in protein function prediction from the fifth CAFA challenge
- Keywords: protein function prediction, CAFA, community benchmark, protein language model
- 简要描述: CAFA5 展示了蛋白语言模型、结构预测和持续生物注释累积，正在系统性提升蛋白功能预测的整体上限，但对已有部分注释蛋白的增量预测仍是难点。
- 研究主题: 社区级蛋白功能预测评测与方法演进
- 背景: 蛋白功能预测长期依赖跨数据集、跨时间的公平评测，否则很难判断新方法究竟带来了多少真实改进。
- 方法: 作者分析 CAFA5 挑战提交结果，并在严格未公开注释与中间版本数据库设定下，对比不同方法在新蛋白和已有不完整注释蛋白上的表现。
- 结果: 论文报告，相比前几轮 CAFA，整体预测精度继续提升；同时新评测设定显示，如何利用已有不完整注释去预测后续新增功能，仍然是现有方法的薄弱环节。
- 意义: 这项工作为蛋白功能预测提供了更可信的进步刻度，也说明未来竞争点不只是更强表示，而是如何把现有知识更有效地纳入增量推断。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.27.716980v1

### 2. TxConformal: Controlling False Discoveries in AI-Driven Therapeutic Discovery
- Original Title: TxConformal: Controlling False Discoveries in AI-Driven Therapeutic Discovery
- Keywords: therapeutic discovery, conformal prediction, false discovery rate, distribution shift
- 简要描述: TxConformal 关注的不是把候选分数再抬高一点，而是让药物发现流程在分布漂移下仍能控制误报，把昂贵实验资源更集中地投向真正有价值的候选。
- 研究主题: 面向治疗发现的可信短名单筛选
- 背景: AI 药物发现经常面对跨模态、跨阶段和跨分布的数据漂移，若只按分数排序，容易在后续实验中付出高额假阳性成本。
- 方法: 作者先用隐藏表示平衡来校正分布漂移，再输出可与多重检验结合的置信度或 p 值，以支持固定错误率约束下的候选选择。
- 结果: 在六个真实任务中，TxConformal 都能稳定控制错误率；在组合遗传扰动筛选中显著减少误报，在蛋白突变结构筛选中则在严格阈值下保留更多高价值候选。
- 意义: 这项工作把“AI 发现”推进成“可控风险的决策过程”，对需要高实验成本验证的生物医药场景尤其关键。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.27.721076v1

### 3. linearPOA: A parallel, memory-efficient framework for Partial Order Alignment with linear space complexity
- Original Title: linearPOA: A parallel, memory-efficient framework for Partial Order Alignment with linear space complexity
- Keywords: partial order alignment, long reads, memory efficiency, sequence assembly
- 简要描述: linearPOA 试图解决超长读长场景下 POA 内存爆炸的问题，用线性空间复杂度把原本难以落地的超长序列比对重新拉回可执行范围。
- 研究主题: 面向超长读长的内存友好型多序列比对
- 背景: 长读长测序与组装越来越依赖 POA 表达复杂序列关系，但传统二次空间复杂度在 100 kbp 级别读长上已难以承受。
- 方法: 作者采用分治思路改写 POA 计算流程，并实现为 linearPOA 库，用于在保持对齐能力的同时大幅降低内存占用。
- 结果: 论文报告，在 100 kbp 读长设定下，该方法相对若干非启发式基线可节省两个数量级的内存，并仍可支持纠错等基础测序分析任务。
- 意义: 这类工作的重要性不只在速度，而在于让超长读长分析的基础算法真正具备工程可行性。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.27.720899v1

- 背景: 酶动力学参数是机制生物学的核心变量，但真实生理条件下的数据极其稀缺，导致很多代谢与疾病分析仍停留在粗粒度近似。

- 意义: 这项工作让“个体化代谢机制建模”更接近可计算现实，为精准医学中的药物反应与通路分析提供了更细的中间层。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.28.721430v1

- 意义: 这说明 theorem-proving agent 的真正资产不只是当前推理轨迹，而是逐步沉淀出的可迁移知识库。

- 意义: 这说明长上下文能力要真正进入生产环境，关键不是某个算法单点最优，而是算法与内存系统的协同设计。

### 12. Accelerating RL Post-Training Rollouts via System-Integrated Speculative Decoding

### 13. DORA: A Scalable Asynchronous Reinforcement Learning System for Language Model Training
- Original Title: DORA: A Scalable Asynchronous Reinforcement Learning System for Language Model Training
- Keywords: asynchronous RL, rollout orchestration, bounded staleness, LLM training
- 简要描述: DORA 试图解决异步 RL 一直存在的核心矛盾: 既想消除长尾 rollout 带来的流水线气泡，又不能破坏策略一致性与收敛性。
- 研究主题: 面向大模型后训练的异步 RL 系统编排
- 背景: 在 LLM 后训练中，rollout 常占据大部分步时，而长尾轨迹又恰恰对性能提升最重要，导致同步训练效率极低。
- 方法: 作者提出 multi-version streaming rollout，让多个策略版本并行存在，并通过算法与系统协同设计约束轨迹一致性、数据完整性与 staleness。
- 结果: 论文报告该系统在开源基准和大规模工业场景中都显著提升吞吐，同时不破坏收敛，并支撑生成具有竞争力的开源推理模型。
- 意义: 这类工作说明，下一阶段 RL 竞争不只在奖励设计，更在于谁能把训练流水线组织得足够稳定和高效。
- 原文链接: https://arxiv.org/abs/2604.26256
