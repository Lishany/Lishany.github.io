### 1. Integrating the MARTINI2 Coarse-Grained Force Field into HADDOCK3 for Faster Modelling of Large Biomolecular Complexes
- Original Title: Integrating the MARTINI2 Coarse-Grained Force Field into HADDOCK3 for Faster Modelling of Large Biomolecular Complexes
- Keywords: coarse-grained docking, HADDOCK3, MARTINI2, biomolecular complexes
- 简要描述: 这篇工作把 MARTINI2 粗粒化力场正式接入 HADDOCK3，让大规模复合物建模在保持生物学可解释性的同时显著降低计算成本。
- 研究主题: 面向大分子复合物的高效粗粒化对接建模
- 背景: 大型蛋白复合物和蛋白-DNA 体系的采样开销很高，传统全原子流程在大规模系统上往往难以兼顾效率与精度。
- 方法: 作者将 MARTINI2 表示、参数和原子级回转流程嵌入 HADDOCK3 的模块化架构中，并在蛋白-蛋白、蛋白-DNA 基准及若干大体系上评估新流程。
- 结果: 该方案在维持生物学相关精度的前提下明显降低了建模开销，说明粗粒化表示能够稳定支持更大规模的对接采样与评分。
- 意义: 这项工作强化了一个重要判断: 结构生物学工作流的下一步不只是更强模型，也包括更好的表示层与系统集成，使大体系建模真正可规模化。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.25.720800v1

### 2. Unraveling protein conformational plasticity with PROTEUS
- Original Title: Unraveling protein conformational plasticity with PROTEUS
- Keywords: protein dynamics, conformational plasticity, zero-shot scoring, structure foundation model
- 简要描述: PROTEUS 试图在不做分子动力学模拟的情况下，直接从结构预测模型内部表征中估计蛋白构象可塑性。
- 研究主题: 蛋白构象可塑性的零样本预测
- 背景: 构象可塑性决定变构调控、fold switching 与修饰可及性，但现有方法通常依赖昂贵模拟或专门实验数据，难以做到蛋白组尺度。
- 方法: 作者利用 SimpleFold 在去噪轨迹中从“仅序列”到“结构收敛”阶段的嵌入变化，定义 PROTEUS 分数，并与多类柔性蛋白和 1290 条分子动力学轨迹对照验证。
- 结果: PROTEUS 能区分从刚性 scaffold 到结合诱导折叠蛋白的柔性谱系，并在 fold-switch、开放/闭合状态及埋藏磷酸化位点识别上给出较高 AUROC。
- 意义: 这意味着结构基础模型不只是给静态坐标，也可能隐式编码动力学信息，为大规模功能位点筛查和可塑性研究提供了更廉价入口。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.27.721098v1

### 5. Housekeeping Gene Expression Normalization in Transcriptomics Mitigates Data Leakage in Machine Learning Models
- Original Title: Housekeeping Gene Expression Normalization in Transcriptomics Mitigates Data Leakage in Machine Learning Models
- Keywords: transcriptomics, normalization, data leakage, biomarker modeling
- 简要描述: 这篇工作把一个常被忽视但极易污染结果的环节拉到台前: 转录组归一化若选错 housekeeping gene，本身就会把泄漏引入机器学习模型。
- 研究主题: 转录组机器学习中的归一化与泄漏控制
- 背景: 许多 RNA-seq 工作默认使用固定 housekeeping genes 做归一化，但这些基因在治疗或临床条件变化下未必稳定，可能导致模型过拟合和结论失真。
- 方法: 作者在肾移植公开数据上结合变异系数、稳定性和 Gini 系数筛选更稳定的 housekeeping genes，并与常见归一化方案比较。
- 结果: 新筛出的 9 个候选基因能更好控制数据泄漏，提升模型稳健性和跨场景泛化能力，优于 median ratio normalization 和 TMM 等标准方法。
- 意义: 对转录组 ML 来说，这项工作提醒人们，高性能并不总来自更复杂模型，很多时候来自前处理环节是否真正避免了统计污染。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.24.720637v1

- Original Title: Learning to Think from Multiple Thinkers

- 简要描述: 这篇理论工作问了一个很扎实的问题: 当多个“思考者”都给出正确但风格不同的推理链时，模型到底更容易学会推理，还是会被多样性拖慢。

- 背景: 近期大量方法默认更多推理链意味着更好监督，但不同 thinker 的系统性差异未必总是帮助学习。
- 方法: 作者在密码学假设下证明，被动收集的多 thinker CoT 仍可能计算上难学，同时提出一个主动学习算法，使每个 thinker 所需 CoT 数据量与目标精度无关。
- 结果: 理论结果表明，多样化 CoT 不是天然红利，真正关键的是主动式数据选择与少量高价值思维轨迹获取。
- 意义: 这为后续构建推理数据集和蒸馏多来源解题过程提供了更严格的理论框架，而不是继续依赖“多收集一些 CoT”这种经验主义策略。
- 原文链接: https://arxiv.org/abs/2604.24737

### 13. ClawTrace: Cost-Aware Tracing for LLM Agent Skill Distillation
- Original Title: ClawTrace: Cost-Aware Tracing for LLM Agent Skill Distillation
- Keywords: tracing, skill distillation, cost attribution, agent optimization
- 简要描述: ClawTrace 把 agent 轨迹蒸馏里长期缺失的一项信息补上了: 每一步到底花了多少钱，以及这笔成本是否真的必要。

- 背景: 传统多模态肿瘤分层模型通常优化总体性能，却难以在 POLE 突变等关键情形下强制执行指南 override，这在临床上是不可接受的。
- 方法: 作者把系统分成病理、分子和临床 specialist agents，再由治理 agent 基于 Guideline Knowledge Graph 应用确定性硬规则与受约束软推理。
- 结果: 文中报告，在 TCGA-UCEC 上 EndoGov 达到 0.943 accuracy 和 0.973 macro AUC，在触发案例中的 conditional logic-violation rate 为 0.93%；在 CPTAC-UCEC 上，其 accuracy 为 0.842。
- 意义: 这项工作说明，真正可落地的医疗多 agent 系统不能只追求分类分数，而必须把可执行知识治理直接写进架构层。
