### 1. Building computational benchmarks: an Omnibenchmark reimplementation of a single-cell preprocessing pipeline evaluation
- Original Title: Building computational benchmarks: an Omnibenchmark reimplementation of a single-cell preprocessing pipeline evaluation
- Keywords: single-cell RNA-seq, benchmark, reproducibility, pipeline evaluation

### 2. DOMINO: Learning Domain Co-occurrence for Multidomain Protein Design
- Original Title: DOMINO: Learning Domain Co-occurrence for Multidomain Protein Design
- Keywords: protein design, multidomain proteins, domain co-occurrence, sequence generation
- 简要描述: DOMINO 试图把自然界中结构域共现模式变成多结构域蛋白设计先验。

- 背景: 多结构域蛋白来自结构域的复用与重组，但自然架构只是可行 domain-combination 空间中的稀疏样本。

- 结果: 论文报告，该框架既能恢复自然 domain pair，也能实现新检索 pair 的序列设计，并生成大量具有高域恢复度和高 AlphaFold 结构置信度的样本。
- 意义: 该工作提供了一种探索新型 multidomain protein architecture 的可扩展策略。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.05.01.721929v1

### 7. OpenSeeker-v2: Pushing the Limits of Search Agents with Informative and High-Difficulty Trajectories
- Original Title: OpenSeeker-v2: Pushing the Limits of Search Agents with Informative and High-Difficulty Trajectories
- Keywords: search agents, supervised fine-tuning, trajectory synthesis, tool use
- 简要描述: OpenSeeker-v2 讨论的是在 search agent 训练里，信息量高且难度高的 trajectory 是否足以让简单 SFT 变得很强。
- 研究主题: search agent 的高信息量轨迹训练
- 背景: 现有 frontier search agent 的训练通常依赖 pre-training、CPT、SFT 和 RL 组成的高资源流水线。
- 方法: 作者围绕 richer knowledge graph、larger tool set 和 strict low-step filtering 做了三项数据合成修改，并只用 SFT 训练 search agent。
- 结果: 论文报告，OpenSeeker-v2 在多个 benchmark 上取得强表现，并将其定位为纯学术团队用 SFT 做出的强 search agent 基线。
- 意义: 该工作提供了一条更轻量的 search agent 训练路径。
- 原文链接: https://arxiv.org/abs/2605.04036

### 8. Correct Is Not Enough: Training Reasoning Planners with Executor-Grounded Rewards
- Original Title: Correct Is Not Enough: Training Reasoning Planners with Executor-Grounded Rewards
- Keywords: reasoning planner, executor-grounded reward, RL, intermediate traces
- 简要描述: TraceLift 把 reasoning trace 当作要被下游 executor 消费的中间产物，而不只看 final answer 对不对。
- 研究主题: reasoning planner 的 executor-grounded reward
- 背景: verifiable reward 常用于强化显式推理，但只看 final correctness 无法说明 reasoning trace 是否 faithful、reliable 或 useful。
- 方法: 作者提出 TraceLift，用 frozen executor 把 planner 生成的 reasoning 转成最终产物，并用 executor-grounded reward 训练 reasoning planner。
- 结果: 论文表明，这种 executor-grounded reward 能提升 planner-executor 两阶段系统，相比只看 execution-only training 更强调 reasoning trace 的可用性。
- 意义: 该工作把 reasoning supervision 从“答案对不对”推进到“推理是否真的帮助后续执行”。
- 原文链接: https://arxiv.org/abs/2605.03862

### 10. MEMTIER: Tiered Memory Architecture and Retrieval Bottleneck Analysis for Long-Running Autonomous AI Agents
- Original Title: MEMTIER: Tiered Memory Architecture and Retrieval Bottleneck Analysis for Long-Running Autonomous AI Agents
- Keywords: long-running agents, tiered memory, retrieval, memory coherence
- 简要描述: MEMTIER 关注的是长时间运行 agent 的 memory coherence 问题，以及 flat-file memory 的瓶颈。
- 研究主题: 长时 autonomous agent 的分层记忆架构
- 背景: 长时 autonomous agent 往往会在多日运行中出现 tool-execution success rate 下降和 memory coherence 退化。
- 方法: 作者提出 tripartite memory architecture，结合 episodic store、加权检索、cognitive weight update、异步 consolidation 与 retrieval policy adaptation。
- 结果: 论文将 MEMTIER 描述为一套用于分析与改善 long-running agent memory bottleneck 的分层框架。
- 意义: 该工作提供了一种把 memory storage、retrieval 和 consolidation 分层处理的 agent memory 方案。

- 简要描述: Terminus-4B 讨论小模型是否能作为 specialized subagent 承担 agentic terminal execution。

### 14. What Happens Inside Agent Memory? Circuit Analysis from Emergence to Diagnosis
- Original Title: What Happens Inside Agent Memory? Circuit Analysis from Emergence to Diagnosis
- Keywords: agent memory, circuit analysis, emergence, diagnosis
- 简要描述: 这篇论文不是只看 agent memory 的外部 write-manage-read pipeline，而是追踪内部 computation circuit 如何出现。
- 研究主题: agent memory 的内部电路分析与故障定位
- 背景: agent memory failure 往往是 silent 的，模型可以生成流畅回答，但在跨 session 的信息提取、保留和检索上已经失败。
- 方法: 作者跨模型规模与 memory framework 追踪 internal feature circuit，分析 write、manage 和 read 阶段的内部实现差异。
- 结果: 论文报告，内部 circuit 的分离可用于对 otherwise silent 的 memory failure 做 stage-level localization。
- 意义: 该工作为 agent memory failure 的内部诊断提供了 circuit-level 视角。
- 原文链接: https://arxiv.org/abs/2605.03354
