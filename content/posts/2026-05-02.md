### 1. SpatialBench: Comparative cross-platform benchmarking of high-resolution spatial transcriptomics using matched mouse lymphoid tissue
- Original Title: SpatialBench: Comparative cross-platform benchmarking of high-resolution spatial transcriptomics using matched mouse lymphoid tissue
- Keywords: spatial transcriptomics, benchmarking, Visium HD, Xenium, MERSCOPE
- 简要描述: SpatialBench 用同一生物学体系下的多平台匹配数据，回答了高分辨率空间转录组到底在哪些层面真正一致、又在哪些层面分辨率差异会改变结论。
- 研究主题: 高分辨率空间转录组的跨平台生物学评测
- 背景: 空间转录组平台快速增多，但现有比较往往停留在技术指标，缺少能把平台差异和真实生物学信号放在同一基准中检验的资源。
- 方法: 作者构建 SpatialBench，整合 Visium HD、Xenium、MERSCOPE 及单细胞、单核参考数据，并在疟原虫挑战小鼠脾脏和 B 细胞特异 Tbx21 knockout 模型中系统比较技术与生物学读出。
- 结果: 论文报告，几大平台都能稳定恢复免疫组织结构与 Tbx21 相关程序，但在可达生物分辨率上存在显著差异；Visium HD 与 Xenium 能分辨生发中心明暗区，而 MERSCOPE 未能解析这一层级结构。
- 意义: 这项工作给空间转录组方法评估提供了更可信的公共参照，也提醒研究者平台选择会直接影响可观察到的组织层次与机制结论。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.29.721531v1

### 2. Reconstructing True 3D Spatial Omics at Single-Cell Resolution
- Original Title: Reconstructing True 3D Spatial Omics at Single-Cell Resolution
- Keywords: 3D spatial omics, optimal transport, flow matching, tissue reconstruction
- 简要描述: DeepSpatial 试图摆脱传统 2.5D 切片堆叠的限制，把空间组学重建推进到连续、可任意深度采样的真实 3D 表达空间。
- 研究主题: 单细胞级 3D 空间组学重建
- 背景: 组织切片会打断真实拓扑，而现有 2.5D 重建方法难以恢复连续深度梯度和完整微环境结构，这已成为 3D 空间组学的核心瓶颈。
- 方法: 作者提出基于 Optimal Transport flow matching 的 DeepSpatial，将组织演化建模为连续动态向量场，并通过求解概率流 ODE 来恢复任意深度的连续组织状态。
- 结果: 论文报告，DeepSpatial 在多模态数据上都优于 2.5D 方案，既能在乳腺癌和头颈癌样本中得到更符合真实微环境的 3D 重建，也能在小鼠脑数据上重建 3900 万细胞规模的连续图谱。
- 意义: 这类方法的重要性在于，它把 3D 空间组学从“切片拼图”推进成连续场重建问题，为后续组织建模和微环境分析提供了更自然的表示。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.28.721395v1

### 3. Physics-inspired computational methods for spatial transcriptomics reveal a dysplasia-restricted premalignant basin and a density-asymmetric autocrine niche in oral mucosal carcinogenesis
- Original Title: Physics-inspired computational methods for spatial transcriptomics reveal a dysplasia-restricted premalignant basin and a density-asymmetric autocrine niche in oral mucosal carcinogenesis
- Keywords: spatial transcriptomics, physical priors, ligand diffusion, Hawkes model, premalignant niche
- 简要描述: 这篇工作把扩散长度、密度稳定性和自激过程等物理先验显式带入空间转录组分析，试图减少“只看共表达”带来的机制幻觉。
- 研究主题: 引入物理先验的空间转录组机制解析
- 背景: 许多空间转录组工具默认细胞通讯和组织状态仅由局部共表达决定，但真实组织过程同时受到扩散、密度与空间招募等物理约束。
- 方法: 作者提出 DC3、基于对数密度景观的状态分析，以及分离背景密度与自激作用的 spatial Hawkes 模型，并将其应用于覆盖正常到 dysplasia 的 32.7 万细胞口腔黏膜癌变图谱。
- 结果: 论文报告，这套方法识别出一个局限于 dysplasia 的异常 basal differentiation 表型，并提示某些看似显著的免疫衰减现象更像截面观察而非稳健机制。
- 意义: 这项工作说明，空间组学分析若不显式编码物理先验，容易把组织结构噪声误读为生物机制，而轻量但可解释的物理模型可以显著提高推断可信度。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.27.721026v1

### 4. Single-cell foundation models reveal context-sensitive cancer programmes under subtype shift
- Original Title: Single-cell foundation models reveal context-sensitive cancer programmes under subtype shift
- Keywords: single-cell foundation models, subtype shift, cancer programmes, interpretability
- 简要描述: 这篇论文表明，单细胞 foundation model 的价值可能不在零样本统一优势，而在经过适配后能更好跨越罕见癌症 subtype 的分布偏移。
- 研究主题: 单细胞 foundation model 在癌症 subtype shift 下的泛化
- 背景: 近期零样本评测让一些人怀疑单细胞 foundation model 是否真的优于简单基线，但这也可能只是因为没有进入真正需要迁移适配的任务设定。
- 方法: 作者对 Geneformer 和 scGPT 在肾、肺、乳腺肿瘤常见 subtype 上进行微调，并与 LightGBM 比较其在未见稀有 subtype 上的迁移表现，同时用 Integrated Gradients 和 SHAP 检查是否学到了有生物意义的适配。
- 结果: 论文报告，三类器官内的常见 subtype 任务几乎都能做到接近完美区分，但在 chromophobe RCC、SCLC 和 TNBC 这类罕见 subtype 上，微调后的 foundation model 明显优于 LightGBM。
- 意义: 这些结果提示，单细胞 foundation model 在经过任务适配后，可能更适合处理临床稀有亚型下的分布偏移问题，而不应只用统一的零样本设定评价其价值。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.28.721114v1

### 5. Scan Cluster: A versatile database-independent prediction tool for multi-genome identification of homologous gene clusters.
- Original Title: Scan Cluster: A versatile database-independent prediction tool for multi-genome identification of homologous gene clusters.
- Keywords: gene clusters, genome mining, homologous clusters, BLAST, HMMER
- 简要描述: Scan Cluster 的核心价值是摆脱数据库先验，把同源基因簇发现重新定义为更开放的跨基因组结构搜索问题。
- 研究主题: 数据库无关的同源基因簇识别与比较
- 背景: 许多基因簇发现工具依赖规则库或已知簇数据库，因此更容易找回熟悉对象，却难以系统发现新型或退化变体。
- 方法: 作者提出基于 BLAST 或 HMMER 的 Python 工具 Scan Cluster，用于在多基因组中搜索用户定义的保守基因簇，同时兼容基因插入、缺失、倒位和插入序列整合等复杂演化事件。

- 意义: 这类工具对 genome mining 的意义在于，它把“找已知簇”扩展为“比较簇如何在多个基因组中演化”，更适合探索未知功能模块。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.29.721675v1

### 6. I Would If I Could: Reasoning about Dynamics of Actions in Multi-Agent Systems
- Original Title: I Would If I Could: Reasoning about Dynamics of Actions in Multi-Agent Systems
- Keywords: multi-agent systems, ATL-D, dynamic actions, epistemic logic
- 简要描述: 这篇论文关注的是多 agent 系统里一个常被忽略的问题: agent 的可行动作不是静态给定的，而是会被授予、撤销，并影响其知识状态。
- 研究主题: 多 agent 系统中的动态动作逻辑
- 背景: 经典 strategic logics 能描述 agent 的策略依赖，但对动作权限本身的动态变化以及 agent 对这些变化的认知处理仍相对薄弱。
- 方法: 作者提出 ATL-D 及其扩展 ATEL-D，用于同时刻画动作授予/撤销过程和相关知识更新，并分析其表达能力、与规范系统的关系以及相关计算复杂度。
- 结果: 论文给出了一套可形式化推理动态动作约束的逻辑框架，为权限变化与知识变化的联动分析提供了理论基础。
- 意义: 这项工作对 agent governance 很关键，因为真实系统中的很多安全和协作问题，恰恰来自“谁在什么时候可以做什么”这一层没有被显式建模。
- 原文链接: https://arxiv.org/abs/2604.26053

### 7. Collaborative Agent Reasoning Engineering (CARE): A Three-Party Design Methodology for Systematically Engineering AI Agents with Subject Matter Experts, Developers, and Helper Agents
- Original Title: Collaborative Agent Reasoning Engineering (CARE): A Three-Party Design Methodology for Systematically Engineering AI Agents with Subject Matter Experts, Developers, and Helper Agents
- Keywords: agent engineering, methodology, helper agents, scientific domains, verification
- 简要描述: CARE 试图把 agent 开发从试错式 prompt 调参，推进成带阶段门、可复核 artifact 和明确责任分工的工程方法学。
- 研究主题: 面向科学领域的系统化 agent 工程
- 背景: 科学场景中的 agent 往往同时需要领域知识、工具编排和验证约束，但很多开发流程仍缺少可审查、可移交的中间工件与阶段化方法。
- 方法: 作者提出由领域专家、开发者和 LLM helper agents 组成的三方工作流，通过 interaction requirements、reasoning policies 和 evaluation criteria 等 artifact 驱动 agent 设计与验收。
- 结果: 论文报告，在一个科学应用案例中，CARE 这类阶段门和 artifact 驱动流程提升了开发效率与复杂查询表现。
- 意义: 这项工作表明，agent 开发流程本身可以被结构化为可复用、可维护、可验证的工程对象，而不只是依赖经验性试错。
- 原文链接: https://arxiv.org/abs/2604.28043

### 8. CareGuardAI: Context-Aware Multi-Agent Guardrails for Clinical Safety & Hallucination Mitigation in Patient-Facing LLMs
- Original Title: CareGuardAI: Context-Aware Multi-Agent Guardrails for Clinical Safety & Hallucination Mitigation in Patient-Facing LLMs
- Keywords: clinical safety, hallucination mitigation, guardrails, patient-facing LLMs
- 简要描述: CareGuardAI 的重点不是让医疗问答模型更会说，而是让系统能结合病人上下文对临床风险和事实风险做双重把关。
- 研究主题: 面向患者交互场景的多 agent 临床安全 guardrails
- 背景: 医疗 LLM 回答常会出现“表面正确但临床上不合适”的问题，尤其当用户上下文不完整时，单纯依赖通用事实校验并不够。
- 方法: 作者提出由 controller、受安全约束的生成模块和双风险评估组成的多阶段推理流程，分别用 Clinical Safety Risk Assessment 和 Hallucination Risk Assessment 控制输出。
- 结果: 论文报告，CareGuardAI 在 PatientSafeBench、MedSafetyBench 和 MedHallu 上持续优于强基线，并把响应发布限制在双风险评分都足够低的条件下。
- 意义: 这类系统说明医疗 agent 的关键不只是回答能力，而是能否在推理时主动管理风险和上下文不确定性。
- 原文链接: https://arxiv.org/abs/2604.26959

### 9. Beyond Single-Agent Alignment: Preventing Context-Fragmented Violations in Multi-Agent Systems
- Original Title: Beyond Single-Agent Alignment: Preventing Context-Fragmented Violations in Multi-Agent Systems
- Keywords: multi-agent security, distributed enforcement, context fragmentation, policy verification
- 简要描述: 这篇论文指出，多 agent 系统中的很多违规不是某个 agent 明显作恶，而是多个局部看来合理的动作跨上下文拼接后共同越界。
- 研究主题: 多 agent 系统中的跨上下文违规防护
- 背景: 现有 prompt 对齐和单点拦截器更适合处理单一上下文中的违规，而多部门、多私有上下文环境中的策略冲突更容易漏检。
- 方法: 作者提出 Distributed Sentinel 和 Semantic Taint Token 协议，在不暴露原始跨域数据的前提下传播安全状态，并通过 Counterfactual Graph Simulation 做跨域策略验证。
- 结果: 论文报告，在 PhantomEcosystem 基准上，Distributed Sentinel 相比 prompt-based filtering 和 rule-based DLP 取得了更高 F1，同时保持较低端到端延迟；作者也显示前沿模型在多 agent 跨域执行流程中仍存在显著违规率。
- 意义: 这项工作把 multi-agent alignment 从“每个 agent 各自合规”推进到“全局交互链路是否合规”的层面，更接近真实组织环境中的安全需求。
- 原文链接: https://arxiv.org/abs/2604.22879

### 10. Modeling Clinical Concern Trajectories in Language Model Agents
- Original Title: Modeling Clinical Concern Trajectories in Language Model Agents
- Keywords: clinical agents, risk trajectories, escalation pressure, human-in-the-loop
- 简要描述: 这篇论文关注的不是让 agent 决定诊疗，而是让它把风险累积过程变得可见，使人类能够在正式升级前更早理解“担忧正在上升”。
- 研究主题: 临床场景中 agent 风险轨迹的显式建模
- 背景: 许多临床 agent 的行为接近阈值触发器，只有到了某一瞬间才突然升级，缺少对前期风险积累过程的可解释呈现。
- 方法: 作者将无记忆风险编码器与一阶、二阶动态系统耦合，构建持续变化的 escalation pressure 信号，并在合成病房场景中比较其与无状态 agent 的行为差异。
- 结果: 论文报告，二阶动态能够在不明显改变升级时点的情况下，暴露更平滑、更具前瞻性的 concern trajectory，便于 human-in-the-loop 监控。
- 意义: 这说明在高风险领域，agent 的价值不一定是代替决策，而可能是把原本不可见的中间风险状态显式化。
- 原文链接: https://arxiv.org/abs/2604.27872

### 11. Exploring Interaction Paradigms for LLM Agents in Scientific Visualization
- Original Title: Exploring Interaction Paradigms for LLM Agents in Scientific Visualization
- Keywords: scientific visualization, coding agents, tool-using agents, persistent memory, interaction paradigms
- 简要描述: 这项工作系统比较了在科学可视化任务中，不同 agent 交互范式到底各擅长什么，而不是笼统地讨论“哪类 agent 更强”。
- 研究主题: 科学可视化中的 agent 交互范式比较
- 背景: 用自然语言驱动可视化工作流已成为 agent 的典型应用，但 structured tool use、computer-use agent 和 coding agent 之间的能力边界并不清楚。
- 方法: 作者在 15 个 SciVis 基准任务上评测 8 类 agent，比较其任务成功率、效率、鲁棒性和算力成本，同时分析 CLI、GUI、API/MCP 和持久记忆等交互因素。
- 结果: 论文报告，通用 coding agents 成功率最高但成本昂贵；领域专用 agents 更稳定高效但灵活性受限；computer-use agents 在单步任务表现不错，却在长流程中受限于规划能力。
- 意义: 这些结果提示，SciVis agent 设计需要在 structured tool use、交互能力和记忆机制之间做组合权衡，而不是假设单一交互范式就足够覆盖全部任务。
- 原文链接: https://arxiv.org/abs/2604.27996

### 12. MCPHunt: An Evaluation Framework for Cross-Boundary Data Propagation in Multi-Server MCP Agents
- Original Title: MCPHunt: An Evaluation Framework for Cross-Boundary Data Propagation in Multi-Server MCP Agents
- Keywords: MCP, multi-server agents, credential propagation, taint tracking, benchmark
- 简要描述: MCPHunt 把多服务器 MCP agent 的风险重新表述为信息流控制问题，重点检验跨边界凭证传播是否会在“看似忠实执行”的过程中自然发生。
- 研究主题: 多服务器 MCP agent 的跨边界数据传播评测
- 背景: 在多 server MCP 工作流中，单个工具的读写权限可能各自无害，但串联后会形成跨域凭证传播，这种结构性风险并不依赖模型主观恶意。
- 方法: 作者提出基于 canary 的 taint tracking、覆盖 risky/benign/hard-negative 条件的环境控制设计，以及区分任务要求传播与策略违规传播的 CRS 分层评测框架。
- 结果: 论文报告，在 3615 条主基准 trace 上，不同模型的策略违规传播率达到 11.5% 到 41.3%，且主要集中在浏览器介导的数据流；prompt 缓解虽有效，但稳定性依赖模型的指令遵循能力。
- 意义: 这项工作把 MCP agent 风险进一步转化为可测、可复现、可按传播机制分解分析的 benchmark 问题。
- 原文链接: https://arxiv.org/abs/2604.27819

### 13. Where Did It Go Wrong? Capability-Oriented Failure Attribution for Vision-and-Language Navigation Agents
- Original Title: Where Did It Go Wrong? Capability-Oriented Failure Attribution for Vision-and-Language Navigation Agents
- Keywords: embodied agents, failure attribution, navigation, capability testing, adaptive test generation
- 简要描述: 这篇论文试图回答 embodied agent 出错时最实际的问题: 失败到底源于感知、记忆、规划还是决策，而不是只给出一个总失败率。
- 研究主题: 视觉语言导航 agent 的能力级失败归因
- 背景: 现有 embodied agent 测试多是系统级指标，虽然能判断是否失败，却难以定位失败来自哪个能力模块，因此对改进指导有限。
- 方法: 作者提出 capability-oriented testing，包括自适应测试样例生成、能力 oracle 和反馈驱动归因机制，用来同时发现失败并把失败映射到具体能力缺陷。
- 结果: 论文报告，该方法能比现有基线发现更多失败案例，并更准确地定位能力级薄弱点，从而给出更可操作的改进方向。
- 意义: 这代表 embodied agent 评测正在从最终成功率，走向更像软件工程中的故障诊断与 root-cause analysis。
- 原文链接: https://arxiv.org/abs/2604.25161
