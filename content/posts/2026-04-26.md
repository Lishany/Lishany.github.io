### 1. H2O: A Foundation Model Bridging Histopathology to Spatial Multi-Omics Profiling
- Original Title: H2O: A Foundation Model Bridging Histopathology to Spatial Multi-Omics Profiling
- Keywords: histopathology, spatial transcriptomics, spatial proteomics, multimodal foundation model
- 简要描述: 这篇工作尝试把常规 H&E 病理切片直接转成空间转录组和空间蛋白组层面的分子图谱，目标是用低成本病理成像替代部分昂贵空间组学测序。
- 研究主题: 病理图像到空间多组学的跨模态预测
- 背景: 空间组学能够解析组织内分子异质性，但成本高、规模化部署困难，而 H&E 图像在临床中高度普及却缺少直接分子信息。
- 方法: 作者提出 H2O，用 Vision Transformer 与大语言模型通过对比学习对齐组织形态与分子语义，在 25 个器官和癌种、130 万对 H&E-空间组学 patch 上训练统一模型。
- 结果: H2O 在多个癌症基准上优于现有方法，并能直接从 H&E 图像中恢复具有生物学意义的 MIF-CD74/CD44 细胞间通讯信号。
- 意义: 这说明常规病理图像有机会成为空间多组学分析的廉价入口，对大规模组织图谱构建和临床转化都很重要。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.21.717342v1

### 2. Turep: Detecting cross-cancer tumor-reactive T cells in single-cell and spatial transcriptomics data
- Original Title: Turep: Detecting cross-cancer tumor-reactive T cells in single-cell and spatial transcriptomics data
- Keywords: tumor-reactive T cells, single-cell RNA-seq, spatial transcriptomics, immunotherapy
- 简要描述: Turep 关注的核心问题是，如何在单细胞和空间转录组数据中稳健地区分真正参与抗肿瘤免疫的 T 细胞与旁观者细胞。
- 研究主题: 跨癌种肿瘤反应性 T 细胞识别
- 背景: 肿瘤浸润淋巴细胞对免疫治疗至关重要，但现有 signature 往往来自单一队列，难以在不同癌种间泛化。
- 方法: 作者整合七种人类恶性肿瘤的单细胞 RNA 与 TCR 配对数据，构建泛癌种 tumor-reactive gene signature，并用生成式数据增强缓解类别不平衡。
- 结果: Turep 在跨癌种评测中达到平均 AUROC 0.870，并且预测出的 tumor-reactive T 细胞比例能够关联免疫治疗响应；扩展到空间组学后，还识别出与抗原呈递增强相关的空间 niche。
- 意义: 这类方法把免疫表型识别从单癌种经验特征推进到更可泛化的跨队列工具，对个体化免疫治疗具有直接价值。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.21.719961v1

### 3. scConcept enables concept-level exploration of single-cell transcriptomic data
- Original Title: scConcept enables concept-level exploration of single-cell transcriptomic data
- Keywords: single-cell, interpretability, topic model, LLM
- 简要描述: scConcept 不再停留在 latent embedding 或 gene program 列表，而是尝试把单细胞表示直接提升到更接近研究者认知单位的“概念层”。
- 研究主题: 单细胞转录组的概念级可解释表示
- 背景: 现有单细胞解释方法往往需要大量事后分析，topic model 生成的程序也容易冗余且难以直接转化为清晰生物概念。
- 方法: 作者将神经 topic model 与大语言模型结合，把基因级主题蒸馏为带标签、描述和基因集定义的概念表示，并映射回单个细胞。
- 结果: 在 16 个单细胞数据集上，scConcept 相比现有方法将聚类性能提升 27.1%，解释性提升 50.7%，还能识别与肿瘤进展和生存相关的程序。
- 意义: 这代表单细胞分析正在从“高维压缩”走向“可直接解释的概念建模”，更接近真实生物学推断流程。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.21.719959v1

- Keywords: RNA-seq, Snakemake, workflow, reproducibility

- 意义: 这类工具型工作虽然不以 flashy 模型为卖点，但对可重复生物信息学分析基础设施非常关键。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.21.719844v1

### 5. Aiki-XP: leakage-controlled multimodal prediction of within-species relative protein expression at pan-bacterial scale
- Original Title: Aiki-XP: leakage-controlled multimodal prediction of within-species relative protein expression at pan-bacterial scale
- Keywords: protein expression, multimodal modeling, bacteria, leakage control
- 简要描述: Aiki-XP 关注的是细菌尺度上更可泛化的蛋白表达预测，并把数据泄漏控制明确当作建模设计的一部分。
- 研究主题: 泛细菌尺度蛋白表达的多模态预测
- 背景: 蛋白表达受多层生物因素共同影响，单一组学视角难以完整刻画，而且训练评测中的泄漏问题会让结果虚高。
- 方法: 作者整合基因组、operon、编码序列、蛋白序列和生物物理特征，在 385 个细菌物种、49 万余基因上构建多层级模型，并采用 leakage-controlled split。
- 结果: 在受控划分下，Aiki-XP 的 Spearman 相关达到 0.592，高于 ESM-C 600M 的 0.509；实验还显示 operon 级上下文比单纯增加蛋白语言模型容量更关键。
- 意义: 这项工作提示人们，真正可迁移的表达预测依赖合理的数据划分和跨尺度上下文，而不仅是更大的 foundation model。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.21.719525v1

### 7. Agentic World Modeling: Foundations, Capabilities, Laws, and Beyond
- Original Title: Agentic World Modeling: Foundations, Capabilities, Laws, and Beyond
- Keywords: world model, agents, taxonomy, evaluation
- 简要描述: 这是一篇面向 agent world model 的大综述，试图把不同社区对 world model 的理解重新放进统一框架中。
- 研究主题: 面向 agent 的 world modeling 统一分类与研究路线图
- 背景: 当 AI 系统从生成文本转向长期交互和任务执行后，环境动态建模逐渐成为瓶颈，但不同子领域对 world model 的定义并不一致。
- 方法: 作者提出“levels × laws”分类法，把能力层分为 Predictor、Simulator、Evolver 三层，并将约束环境分为物理、数字、社会和科学四类。
- 结果: 论文系统梳理了 400 余篇工作、100 余个代表性系统，总结了不同层级和法则组合下的方法、失败模式和评测实践。
- 意义: 这类框架型工作的重要性在于，它为 agent world model 提供了可讨论、可比较的共同语言，也把评测重心拉向决策相关能力而非纯预测分数。
- 原文链接: https://arxiv.org/abs/2604.22748

### 8. Seeing the Whole Elephant: A Benchmark for Failure Attribution in LLM-based Multi-Agent Systems
- Original Title: Seeing the Whole Elephant: A Benchmark for Failure Attribution in LLM-based Multi-Agent Systems
- Keywords: multi-agent systems, failure attribution, debugging, benchmark
- 简要描述: 这篇论文聚焦多智能体系统里一个非常实际却长期缺 benchmark 的问题: 失败之后，到底该怪哪个 agent、哪一步。
- 研究主题: 多智能体系统失败归因基准
- 背景: 多 agent 系统的输出是自然语言且交互复杂，传统只看输出的 trace 很难支持真实调试场景中的失败定位。
- 方法: 作者提出 TraceElephant，在可复现实验环境下提供完整执行轨迹，强调 full execution observability 而不是只保留部分输出。
- 结果: 论文报告，与部分观测版本相比，完整轨迹在其基准中可将失败归因准确率提高最多 76%，说明遗漏输入与上下文会掩盖大量真实故障原因。
- 意义: 这把 agent debugging 从模糊经验问题推进为可基准化研究对象，对后续可解释 agent 工具链很关键。
- 原文链接: https://arxiv.org/abs/2604.22708

### 9. Superminds Test: Actively Evaluating Collective Intelligence of Agent Society via Probing Agents
- Original Title: Superminds Test: Actively Evaluating Collective Intelligence of Agent Society via Probing Agents
- Keywords: agent society, collective intelligence, evaluation, probing
- 简要描述: 这篇论文不再默认“大量 agent 聚在一起就会自然涌现群体智能”，而是直接去测它是否真的发生。
- 研究主题: 大规模 agent society 的集体智能评测
- 背景: 随着 agent population 扩大到百万级，社区普遍期待 collective intelligence 涌现，但系统性实证评测仍然缺失。
- 方法: 作者在拥有两百多万个 agent 的 MoltBook 平台上，引入分层 probing agent 框架，从联合推理、信息综合和基础交互三个层次评测群体智能。
- 结果: 实验显示该 agent society 在复杂推理上并未超过单个前沿模型，信息整合和简单协作任务也表现较弱，平台级交互普遍浅且稀疏。
- 意义: 论文给出的结论很直接，规模本身不足以产生 collective intelligence，真正的瓶颈在于交互深度与信息传递结构。
- 原文链接: https://arxiv.org/abs/2604.22452

- Original Title: From Skills to Talent: Organising Heterogeneous Agents as a Real-World Company
- Keywords: multi-agent organization, talent market, orchestration, tree search

- 意义: 这说明多 agent 系统的上限不只受单体能力约束，还取决于组织接口、人才调度和反馈闭环如何设计。

### 11. Memanto: Typed Semantic Memory with Information-Theoretic Retrieval for Long-Horizon Agents
- Original Title: Memanto: Typed Semantic Memory with Information-Theoretic Retrieval for Long-Horizon Agents
- Keywords: memory, long-horizon agents, semantic retrieval, typed schema
- 简要描述: Memanto 关注的是长程 agent memory 的工程瓶颈，尤其是图结构记忆系统在写入和检索时的额外复杂度。
- 研究主题: 长程 agent 的类型化语义记忆层
- 背景: 许多记忆系统依赖混合图和向量结构，需要实体抽取、图 schema 维护和多步检索，部署成本高。
- 方法: 作者提出包含 13 类预定义记忆类型的语义记忆 schema，加入冲突消解与时间版本控制，并用信息论检索引擎实现无索引、低延迟检索。
- 结果: 论文报告在 LongMemEval 和 LoCoMo 上，Memanto 分别达到 89.8% 和 87.1% 准确率，超过文中比较的图和向量混合系统，同时保持单次检索与零写入延迟优势。
- 意义: 这类工作表明，agent memory 的关键不一定是更复杂的图，而可能是更清晰的 typed schema 与更低开销的检索机制。
- 原文链接: https://arxiv.org/abs/2604.22085

### 12. When Does LLM Self-Correction Help? A Control-Theoretic Markov Diagnostic and Verify-First Intervention
- Original Title: When Does LLM Self-Correction Help? A Control-Theoretic Markov Diagnostic and Verify-First Intervention
- Keywords: self-correction, control theory, Markov model, verification
- 简要描述: 这篇论文把 LLM 自我修正从经验技巧变成了一个可以被诊断和干预的控制问题。
- 研究主题: 自校正何时有效的可诊断条件
- 背景: 迭代式 self-correction 已广泛用于 agentic LLM 系统，但它到底何时提升、何时恶化结果，过去缺少简单可操作的判据。
- 方法: 作者用两状态 Markov 模型刻画 Correct 与 Incorrect 之间的转移，并提出基于错误保持率和错误引入率的部署诊断条件，同时测试 verify-first 提示干预。
- 结果: 文中在 7 个模型和 3 个数据集上的实验显示，接近零的错误引入阈值决定自校正是有益还是有害；对 GPT-4o-mini，verify-first 提示在该实验中把 -6.2 个百分点退化转成 +0.2 个百分点增益。
- 意义: 这项研究的价值在于，它提示工程实践中不应把 self-correction 当默认选项，而应根据可观测误差动力学决定是否启用。
- 原文链接: https://arxiv.org/abs/2604.22273

### 13. Sound Agentic Science Requires Adversarial Experiments
- Original Title: Sound Agentic Science Requires Adversarial Experiments
- Keywords: scientific agents, falsification, adversarial experiments, methodology
- 简要描述: 这篇短文提出了一个很重要的方法学提醒: agent 能加速分析，也同样能加速“看起来合理但未经真正检验”的结论生产。
- 研究主题: agentic science 的反证优先原则
- 背景: 当 LLM agent 被用于科学分析时，系统很容易通过选择性分析和不断修改叙述来快速制造可发表的正结果。
- 方法: 作者主张对非实验性、由 agent 辅助生成的科学结论采用 falsification-first 标准，即优先搜索结论会在哪里失败，而不是如何讲得更有说服力。
- 结果: 论文没有给出大型 benchmark，而是从科学方法论角度明确指出，缺失的反证实验正是当前 agentic science 最危险的空白。
- 意义: 这篇文章的重要性不在于提出新模型，而在于为 agent 参与科学研究设定了更严格的验证基线。
- 原文链接: https://arxiv.org/abs/2604.22080

### 14. Contexts are Never Long Enough: Structured Reasoning for Scalable Question Answering over Long Document Sets
- Original Title: Contexts are Never Long Enough: Structured Reasoning for Scalable Question Answering over Long Document Sets
- Keywords: long-context QA, structured reasoning, relational database, evidence synthesis
- 简要描述: 这篇工作直指长文档问答中的一个根本问题: 上下文窗口再长，也终究会被不断增长的证据集合吃满。
- 研究主题: 超长文档集合上的结构化问答
- 背景: 常见做法是分块检索再汇总，但随着 chunk 数增长，系统仍需要面对越来越大的证据聚合瓶颈。
- 方法: 作者提出 SLIDERS，把文档中的关键信息抽取到关系数据库中，用 SQL 在持久结构化状态上推理，并加入数据 reconciliation 阶段修复重复、冲突和缺失记录。
- 结果: 论文报告 SLIDERS 在三个现有长上下文基准上均超过文中比较基线，平均高于 GPT-4.1 6.6 分，并在 390 万和 3600 万 token 的两个新基准上领先次优方法。
- 意义: 这项工作提醒人们，长上下文问题的可扩展解法未必是继续拼更长窗口，而可能是把推理对象从文本流切换为结构化状态。
- 原文链接: https://arxiv.org/abs/2604.22294
