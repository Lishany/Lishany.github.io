### 1. AI-guided discovery of atypical protein assemblies
- Original Title: AI-guided discovery of atypical protein assemblies
- Keywords: structural biology, AlphaFold 3, protein assembly, novelty discovery
- 简要描述: 这篇工作使用 AI 去筛查偏离经典架构的蛋白复合体。
- 研究主题: 异常蛋白复合体的 AI 引导发现
- 背景: AI 系统已经显著推动了蛋白结构预测，但用于发现新的大分子复合体类型的能力仍有待系统开发。

- 意义: 该工作提供了一个用于发现 atypical protein complex 的可扩展筛查框架。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.05.03.722499v1

### 2. DPLM: Dynamics-aware Protein Language Model via contrastive learning between sequence and molecular dynamics simulation trajectory

- 结果: 论文显示，DPLM 能捕捉与突变效应、稳定性和无序区识别相关的动态属性。
- 意义: 该工作提供了一种把 MD 轨迹信息引入蛋白语言模型表示学习的路径。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.29.721692v1

### 7. To Call or Not to Call: A Framework to Assess and Optimize LLM Tool Calling
- Original Title: To Call or Not to Call: A Framework to Assess and Optimize LLM Tool Calling
- Keywords: tool calling, decision theory, web search, hidden-state controller
- 简要描述: 这篇论文把 agent 最常见的一个动作拆开来看: 真正难的可能不是“怎么调用工具”，而是“到底该不该调用”。
- 研究主题: LLM 工具调用的决策评估与优化
- 背景: 对 web search 这类工具而言，工具使用并不总是有益，调用收益取决于模型已有知识与外部信息质量。
- 方法: 作者从 necessity、utility 和 affordability 三个维度定义 tool-use 决策，并训练基于 hidden state 的控制器来优化调用判断。
- 结果: 论文发现，模型的工具使用判断并不总与最优需求一致，而控制器可以改善 decision quality。
- 意义: 该工作提供了一个面向 tool calling 的决策论框架。
- 原文链接: https://arxiv.org/abs/2605.00737

### 8. AEM: Adaptive Entropy Modulation for Multi-Turn Agentic Reinforcement Learning
- Original Title: AEM: Adaptive Entropy Modulation for Multi-Turn Agentic Reinforcement Learning
- Keywords: agentic RL, entropy modulation, credit assignment, exploration-exploitation
- 简要描述: AEM 通过调节训练中的熵动态来改进多轮 agent 的探索行为。
- 研究主题: 多轮 agent RL 的无监督 credit assignment
- 背景: 多轮 agent RL 常只提供稀疏的 outcome-only reward，因此很难把回报分配到具体行动步骤。
- 方法: 作者围绕 response-level entropy dynamics 设计一种 supervision-free credit assignment 方式，以调节探索与利用平衡。
- 结果: 论文报告，AEM 能在多轮 agent RL 中改善探索与利用之间的平衡。
- 意义: 该工作提供了一种 supervision-free 的多轮 agent RL credit assignment 方法。
- 原文链接: https://arxiv.org/abs/2605.00425

### 9. AgentFloor: How Far Up the tool use Ladder Can Small Open-Weight Models Go?
- Original Title: AgentFloor: How Far Up the tool use Ladder Can Small Open-Weight Models Go?
- Keywords: open-weight models, benchmark, tool use, long-horizon planning
- 简要描述: AgentFloor 关心的是工程上非常实际的问题: agent pipeline 里究竟哪些环节必须用大模型，哪些环节其实可以被便宜模型接住。
- 研究主题: 小型开源模型在 agent 工具链中的能力边界
- 背景: 现实 agent 系统包含大量短而结构化的模型调用，因此需要判断哪些步骤真正需要大模型能力。
- 方法: 作者构建一个聚焦工具使用与规划能力的 30 任务基准，用来比较 open-weight model 在不同 agent 子任务上的表现。
- 结果: 结果显示，小中型开源模型已能覆盖相当多的短程结构化工具使用，而长期规划和持续约束维护仍明显更困难。
- 意义: 该工作提供了一个用于评估模型路由边界的 benchmark。
- 原文链接: https://arxiv.org/abs/2605.00334

### 10. Are Tools All We Need? Unveiling the Tool-Use Tax in LLM Agents
- Original Title: Are Tools All We Need? Unveiling the Tool-Use Tax in LLM Agents
- Keywords: tool-use tax, semantic distractors, inference-time gating, reasoning
- 简要描述: 这篇论文对“加工具一定更强”这个流行假设做了逆向检查，并指出工具协议本身也可能引入性能税。
- 研究主题: LLM agent 中的 tool-use tax
- 背景: tool-augmented reasoning 是 LLM agent 的常见方向，但在 semantic distractor 存在时并不总能优于 native CoT。
- 方法: 作者将工具使用中的提示格式、协议开销和实际执行收益拆开分析，以识别 tool-use tax 的来源。
- 结果: 论文表明，在语义噪声条件下，工具带来的收益未必足以抵消 tool-calling protocol 的额外开销。
- 意义: 该工作将 protocol overhead 明确为工具增强系统中的独立误差来源。
- 原文链接: https://arxiv.org/abs/2605.00136

### 11. Position: agentic AI orchestration should be Bayes-consistent
- Original Title: Position: agentic AI orchestration should be Bayes-consistent
- Keywords: Bayesian decision theory, orchestration, uncertainty, control layer
- 简要描述: 这不是一篇新 benchmark 论文，而是一篇立场明确的 position paper: Bayesian 思想最该落地的位置可能不是模型参数，而是 agent 控制层。
- 研究主题: agent 编排层的 Bayesian 一致性
- 背景: 高价值 agent 部署往往涉及不确定条件下的决策，例如工具选择、专家选择和资源投入。
- 方法: 作者从 Bayesian decision theory 出发，主张 orchestrator 应维护对潜在变量的信念、依据交互更新这些信念，并以效用敏感方式选择动作。
- 结果: 文章的主要结论是，显式 Bayesian LLM 本身未必现实，但 orchestration layer 是较清晰且可行的 Bayesian 落点。
- 意义: 该工作将 agent orchestration 表述为一个 Bayesian decision 问题。
- 原文链接: https://arxiv.org/abs/2605.00742
