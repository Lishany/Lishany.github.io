### 1. PanVariants: Best Practice for Pangenome-based Variant Calling Pipeline and Framework
- Original Title: PanVariants: Best Practice for Pangenome-based Variant Calling Pipeline and Framework
- Keywords: pangenome, variant calling, structural variants, joint calling
- 简要描述: 这篇工作把 pangenome variant calling 从“只能识别图中已知变异”的半成品状态推进到同时兼顾新变异发现和临床可用性的完整流程。
- 研究主题: 泛基因组参考下的高精度变异检测
- 背景: 线性参考在群体多样性表示上存在局限，而现有 pangenome 方案又常常偏向已知变异检出，难以满足临床和队列分析需求。
- 方法: 作者提出 PanVariants，将 k-mer 驱动和 mapping 驱动方法结合，并系统比较其与 BWA+GATK/Manta、DRAGEN/PanGenie 在小变异、结构变异、阳性样本和 cohort joint calling 场景下的表现。
- 结果: 对 SNV 错误数相对 BWA+GATK 降低 73%，相对 DRAGEN 降低 45%；SV 检测 F1 达到 89.39%，明显高于两类传统方案，并接近长读长表现。
- 意义: 这项工作说明 pangenome reference 已不只是研究概念，而是在真实诊断和群体分析里开始具备替代线性参考的可操作路径。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.22.720142v1

### 2. MSAgent: An Evidence Grounded Agentic Framework for LLM-driven Scientific Exploration in Mass Spectrometry-based Metabolomics
- Original Title: MSAgent: An Evidence Grounded Agentic Framework for LLM-driven Scientific Exploration in Mass Spectrometry-based Metabolomics
- Keywords: mass spectrometry, metabolomics, LLM agent, evidence-grounded workflow
- 简要描述: MSAgent 试图把质谱分子鉴定中高度依赖专家经验和工具拼接的流程，重构成一条带证据链的自治分析系统。
- 研究主题: 面向代谢组学结构解析的科学 agent
- 背景: 质谱数据获取速度远高于结构注释速度，当前分析仍强依赖人工整合多个孤立工具，形成明显吞吐瓶颈。
- 方法: 作者组织 50 多个领域工具构成 MSToolbox，由 LLM 进行意图感知规划、跨资源结果综合和可追踪报告生成，并在 CASMI 与 CANOPUS 等公开基准上评估。
- 结果: 在 CASMI 系列基准中 MRR 提升超过 10%，在更难的 de novo 任务上对基线结果的 Tanimoto 相似度带来 40% 以上平均增益。
- 意义: 这表明 scientific agent 的价值不只是“会调工具”，而是能把专家式推理、证据聚合和质量控制整合成可复查的科学工作流。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.22.720103v1

### 4. CellPulse: A Foundation Model of Coordinated Gene Dynamics Simulating Viral Infectious Diseases
- Original Title: CellPulse: A Foundation Model of Coordinated Gene Dynamics Simulating Viral Infectious Diseases
- Keywords: foundation model, viral infection, bulk RNA-seq, host response
- 简要描述: CellPulse 想解决的是，如何从大规模 bulk 转录组中学习病毒感染下宿主反应的方向性和协调性，而不是只做静态表达模式拟合。
- 研究主题: 病毒感染宿主反应的方向感知基础模型
- 背景: 现有 gene expression foundation model 多依赖单细胞和静态表达，难以直接覆盖临床中更常见的 bulk 干预响应场景。
- 方法: 作者基于 2300 多万条病毒感染差异表达谱构建 VISTA，并用结构化 differential expression 表示和 direction-aware attention 训练 CellPulse。

- 意义: 这说明 bulk transcriptomics 也可以成为学习扰动动力学和治疗靶点发现的 foundation model 土壤，不必完全依赖单细胞范式。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.22.720078v1

- Original Title: MVCBench: A Multimodal Benchmark for Drug-induced Virtual Cell Phenotypes
- Keywords: virtual cell, multimodal benchmark, drug response, representation learning
- 简要描述: MVCBench 聚焦一个经常被忽略的问题: 在药物诱导虚拟细胞任务里，到底什么样的药物表示和基因表示真正有效。
- 研究主题: 药物诱导多模态虚拟细胞的系统基准评测
- 背景: 虽然药物分子表示和基因表示方法发展很快，但在大规模 transcriptomics 与 morphology 联合预测任务上仍缺少统一、严格的比较框架。
- 方法: 作者基于接近 110 万个药物诱导 profile，系统评测 24 种表示方法，并比较其在分布内与分布外场景中的表现。
- 结果: 研究发现高级分子表示对形态表型预测帮助明显，但对基因表达预测增益有限，同时跨数据分布泛化仍明显脆弱。
- 意义: 这项工作把 virtual cell 研究从“堆模型”拉回到表示选择、模态融合和 distribution shift 这些更决定落地效果的核心问题上。

### 6. GenNA: Conditional generation of nucleotide sequences guided by natural-language annotations
- Original Title: GenNA: Conditional generation of nucleotide sequences guided by natural-language annotations
- Keywords: nucleotide generation, multimodal pretraining, natural language guidance, sequence design
- 简要描述: GenNA 把自然语言注释作为高层语义接口，引入到核酸序列生成里，尝试缩短“功能描述”和“序列设计”之间的距离。
- 研究主题: 自然语言驱动的核酸条件生成
- 背景: 现有 nucleotide language model 虽然能学习序列分布，但缺少对功能语义的显式接入，因此难以支持真正的 semantics-guided generation。
- 方法: 作者在覆盖 2221 个真核物种的 nucleotide-text 多模态语料上训练统一模型，使其同时学习序列模式与功能注释关系。
- 结果: 模型能对语义不匹配、突变类型和物种标签扰动给出可区分的 perplexity 响应，并在多类自然语言引导生成任务中产出与语义和物种上下文一致的序列。
- 意义: 这为把高层功能说明直接接到低层序列设计提供了一个可扩展框架，是生物序列设计从纯统计生成走向语义交互的重要一步。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.22.720063v1

### 7. AgentSearchBench: A Benchmark for AI Agent Search in the Wild
- Original Title: AgentSearchBench: A Benchmark for AI Agent Search in the Wild
- Keywords: agent search, retrieval, reranking, execution signals
- 简要描述: 这篇论文关注 agent 生态扩大后的一个基础设施问题: 面对成千上万个 agent，系统该如何找到真正适合某个任务的那一个。
- 研究主题: 真实世界 agent 搜索基准
- 背景: 传统工具搜索更依赖静态功能描述，但 agent 能力通常具有组合性和执行依赖性，仅靠文本描述很难判断真实适配度。
- 方法: 作者从近一万个真实 agent 构建 AgentSearchBench，把 agent 搜索形式化为 retrieval 与 reranking 问题，并用 execution-grounded performance signal 评估相关性。
- 结果: 实验显示语义相似度与实际 agent 表现之间存在稳定落差，说明仅依赖描述文本难以反映真实执行质量。
- 意义: 这意味着未来 agent 平台的核心能力之一将不是造更多 agent，而是建立执行感知的 agent discovery 和 routing 机制。
- 原文链接: https://arxiv.org/abs/2604.22436

### 10. How LLMs Detect and Correct Their Own Errors: The Role of Internal Confidence Signals
- Original Title: How LLMs Detect and Correct Their Own Errors: The Role of Internal Confidence Signals
- Keywords: self-correction, confidence signals, second-order reasoning, mechanistic analysis
- 简要描述: 这篇论文试图回答一个比“self-correction 是否有效”更底层的问题: LLM 到底凭什么知道自己错了，以及什么时候有能力改正。
- 研究主题: LLM 自我纠错的内部置信信号机制
- 背景: 大模型能在没有外部反馈时识别和纠正部分错误，但这类行为究竟来自 log-probability 还是独立评估信号，机制上仍不清楚。
- 方法: 作者借鉴决策神经科学中的 second-order confidence 框架，研究答案生成后的内部置信信号，并在 verify-then-correct 设置下做行为与因果干预分析。
- 结果: 这种内部置信信号不仅能超越 token log-probability 预测错误检测，还能预测哪些错误真正可被修正；在答案信息受损时，因果干预可恢复相关行为。
- 意义: 这提示自我纠错不只是 prompt 技巧，而可能建立在可单独建模的内部评估子系统上，对可解释纠错和 agent control 都有价值。
- 原文链接: https://arxiv.org/abs/2604.22271

### 11. Reliable Self-Harm Risk Screening via Adaptive Multi-Agent LLM Systems
- Original Title: Reliable Self-Harm Risk Screening via Adaptive Multi-Agent LLM Systems
- Keywords: multi-agent systems, safety-critical AI, adaptive sampling, behavioral health
- 简要描述: 这项工作把多 agent LLM 在行为健康风险筛查中的使用，从启发式多数投票推进到有统计保证的决策框架。
- 研究主题: 安全关键场景下的自适应多 agent 判定
- 背景: 在自伤风险和抑郁筛查等任务中，多步 LLM pipeline 很常见，但 LLM-as-a-judge 或启发式 voting 无法说明结论何时可靠，也难追踪错误累积。
- 方法: 作者把 agent pipeline 建模为 DAG 上的随机分类决策系统，引入更紧的 agent-level 置信界、自适应 bandit sampling，以及系统级 regret 保证。
- 结果: 在两个行为健康数据集上，自适应采样在保持相近假阴性率的同时显著减少了误报安全内容。
- 意义: 这篇论文的重要性在于提醒人们，安全关键 LLM 系统不能只看平均分数，更需要显式决策理论和误差控制机制。
- 原文链接: https://arxiv.org/abs/2604.22154
