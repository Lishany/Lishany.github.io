### 1. Deterministic retrieval recovers biomedical associations lost by language models
- Original Title: Deterministic retrieval recovers biomedical associations lost by language models
- Keywords: biomedical retrieval, deterministic graph retrieval, entity resolution, reproducibility
- 简要描述: 这篇工作指出，LLM 检索在生物医学关联发现中会系统性漏掉关系，而 BioChirp 通过把 LLM 解释层与确定性图检索层分离，提升了覆盖率和可重复性。
- 研究主题: 面向生物医学知识发现的混合检索架构
- 背景: 仅依赖 LLM 进行检索时，输出截断、同义词不匹配和多次运行不稳定会导致大量关联缺失，这对高价值生物医学知识发现尤其不利。
- 方法: 作者使用 LLM 负责查询解释和候选过滤，再结合多源共识实体解析与确定性图检索，在四个主要生物医学数据库上进行比较评估。
- 结果: 相比常规 LLM 检索流程，BioChirp 恢复了更多生物医学关联，并且结果在多次运行中更稳定。
- 意义: 这项工作强化了一个重要判断: 在科研检索场景里，LLM 更适合作为语义接口层，而不是取代可验证、可重复的底层检索机制。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.25.720782v1

- 研究主题: 预训练蛋白表示驱动的蛋白功能预测

### 4. A transcriptomic-driven segmentation and cell simulation framework for high-resolution spatial transcriptomics and cell-cell communication

- 简要描述: TENGU 的核心价值在于把空间转录组分割从图像优先改成转录信号优先，从而在高密度、炎症和矿化等复杂组织里减少转录本串扰。
- 研究主题: 高分辨率空间转录组的 transcript-first 分割与细胞模拟

- 意义: 这项工作说明，空间生物学的瓶颈不只是分辨率提升，更在于是否拥有与分辨率匹配的分析管线。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.24.720489v1

### 6. Learning dynamics of unsupervised deep learning reveal epoch-specific genetic architectures of brain morphology
- Original Title: Learning dynamics of unsupervised deep learning reveal epoch-specific genetic architectures of brain morphology
- Keywords: representation learning, imaging genetics, checkpoint ensemble, brain MRI
- 简要描述: 这篇工作把深度模型训练过程本身变成了遗传发现对象，指出不同训练 epoch 学到的表征会对应不同的遗传结构。
- 研究主题: 无监督表示学习动态与脑影像遗传发现
- 背景: 表示学习已广泛用于从复杂影像中提取表型做遗传研究，但以往工作通常只取单个最终 checkpoint，忽略了训练过程中表征不断重组的现象。
- 方法: 作者在 UK Biobank 的 T1 MRI 上训练 3D 卷积自编码器，系统分析不同训练阶段表征与脑形态遗传力、基因位点和神经解剖特征之间的关系。
- 结果: 不同 epoch 在相近遗传力水平下捕捉到不同遗传架构，跨多个信息性 checkpoint 的集成比单 checkpoint 方案能发现更多风险位点。
- 意义: 这提示今后的表示学习型遗传研究不应只关心“最终模型”，还要关注训练轨迹本身蕴含的科学信号。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.24.720631v1

- 意义: 这类工作强调，长程 agent 系统的核心资产不是更长上下文，而是显式、可恢复、可验证的知识状态管理。

### 11. How Fast Should a Model Commit to Supervision? Training Reasoning Models on the Tsallis Loss Continuum
- Original Title: How Fast Should a Model Commit to Supervision? Training Reasoning Models on the Tsallis Loss Continuum
- Keywords: reasoning training, RLVR, Tsallis loss, cold-start mitigation
- 简要描述: 这篇理论与方法结合的工作试图回答一个关键问题: 推理模型在训练时到底应多快从“探索潜在轨迹”转向“压实监督信号”。
- 研究主题: 推理模型后训练中的损失连续体设计
- 背景: 仅依赖可验证奖励的 RLVR 在初始成功率很低时容易陷入冷启动停滞，导致模型学不会进入高价值轨迹区域。
- 方法: 作者基于 Tsallis q-logarithm 构造从 RLVR 到 latent trajectory log-marginal-likelihood 的连续损失家族，并提出 GARL 与 PAFT 两种 Monte Carlo 估计器。

- 意义: 这项工作把“后训练目标函数本身”重新拉回核心位置，说明推理模型能力并不只是由数据和模型大小决定，优化路径同样关键。
- 原文链接: https://arxiv.org/abs/2604.25907

- 意义: 这项工作提醒人们，后训练安全评测不能只看表面平均分，还要主动寻找触发条件和上下文依赖边界。

### 14. Odysseys: Benchmarking Web Agents on Realistic Long Horizon Tasks
- Original Title: Odysseys: Benchmarking Web Agents on Realistic Long Horizon Tasks
- Keywords: web agent benchmark, long-horizon tasks, rubric evaluation, trajectory efficiency
- 简要描述: Odysseys 把 Web agent 评测从短任务刷分拉回真实使用场景，强调多站点、长时间跨度和过程效率才是下一阶段瓶颈。

- 方法: 作者基于真实浏览会话构建 200 个 live Internet 长程任务，引入 rubric-based evaluation 替代简单 pass/fail，并增加按步效率指标来评估 agent 是否“高效地完成任务”。
- 结果: 最强模型的成功率约为 44.5%，Trajectory Efficiency 仅为 1.15%，说明即使强模型也远未达到高效、稳定的长程 Web 操作能力。
- 意义: 这项工作为 computer-use agent 提供了更接近真实部署的新评测方向，尤其强调效率而非最终偶然成功。
