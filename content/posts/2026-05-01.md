### 1. SpatialQuery: scalable discovery and molecular characterization of multicellular motifs from spatial omics data
- Original Title: SpatialQuery: scalable discovery and molecular characterization of multicellular motifs from spatial omics data
- Keywords: spatial omics, multicellular motifs, differential expression, covariation analysis
- 简要描述: SpatialQuery 把空间组学分析从单细胞或成对关系，推进到可系统发现多细胞共定位模式并解析其分子程序的统一框架。
- 研究主题: 空间组学中的多细胞模式发现与分子特征解析
- 背景: 空间转录组和空间蛋白组正在快速扩展，但现有方法通常只覆盖局部邻域或成对细胞关系，难以同时完成模式发现与机制解释。
- 方法: 作者提出 SpatialQuery，用于识别重复出现的多细胞 motif，并围绕这些 motif 进行差异表达与协变分析，从而解析空间环境调制下的分子程序。
- 结果: 论文报告，该方法在空间转录组和蛋白组数据中识别出肠管发育中的跨胚层信号、肾脏和结肠疾病中的纤维化与免疫抑制 niche，以及小鼠脑图谱中的区域性转录调控特征。
- 意义: 这项工作让空间组学分析更接近“从结构到机制”的工作流，也为交互式细胞图谱门户中的大规模分析提供了轻量化基础设施。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.22.720136v2

### 2. Accurate ab initio gene prediction in eukaryotes with Tiberius in multiple clades
- Original Title: Accurate ab initio gene prediction in eukaryotes with Tiberius in multiple clades
- Keywords: gene prediction, eukaryotic genomes, deep learning, genome annotation
- 简要描述: Tiberius 试图把深度学习基因预测从少数谱系扩展到多类真核生物，在保持高精度的同时显著压低大规模注释的运行成本。
- 研究主题: 多谱系真核基因组的 ab initio 基因预测
- 背景: 真核基因组注释正被大规模测序项目推向瓶颈，传统 ab initio 方法在泛化性、速度和准确率之间往往难以兼顾。
- 方法: 作者为 Tiberius 训练了覆盖被子植物、真菌、脊椎动物、昆虫、绿藻和硅藻等多个谱系的深度学习模型，并在 33 个物种上与多类基线进行比较。
- 结果: 论文报告，Tiberius 在整体精度上持续优于 Helixer 和 ANNEVO；相较结合 RNA-Seq 与蛋白证据的 BRAKER3，在多个谱系中已接近最先进水平，同时在 GPU 上平均快约 80 倍。
- 意义: 这说明高通量基因组注释的关键突破，不只是精度再提升一点，而是让跨物种、跨项目的规模化注释真正具备工程可行性。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.24.720536v1

- Keywords: single-cell perturbation, virtual cell, knowledge graph, transformer, LLM embeddings
- 简要描述: scPert 通过把大模型嵌入与结构化生物知识融合，提升了对单基因和组合扰动后单细胞转录响应的预测能力。
- 研究主题: 单细胞遗传扰动效应的多模态预测
- 背景: 虚拟细胞建模被寄望于降低药物发现中的实验成本，但现有方法在复杂遗传互作、可解释性和对未见基因的泛化上仍明显受限。

- 结果: 论文报告，scPert 在单基因和组合扰动任务上都优于已有方法，并在癌症相关分析中揭示了 p53 通路动态和免疫检查点调控机制，同时对 42 个癌症依赖基因给出了潜在治疗靶点线索。
- 意义: 这项工作代表单细胞扰动建模正在从纯数据拟合走向“模型表示 + 生物知识”融合，为虚拟筛选和靶点发现提供了更可信的中间层。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.24.720560v1

### 5. Deterministic retrieval recovers biomedical associations lost by language models
- Original Title: Deterministic retrieval recovers biomedical associations lost by language models
- Keywords: biomedical retrieval, deterministic search, entity resolution, reproducibility, LLM
- 简要描述: BioChirp 关注的不是让语言模型再多记一点知识，而是用确定性图检索与实体对齐机制，把 LLM 检索流程中丢失的生物医学关联重新找回来。
- 研究主题: 面向生物医学知识发现的确定性检索增强
- 背景: 仅依赖 LLM 的检索流程容易受截断、同义词不匹配和运行波动影响，在需要高召回与可复现性的生物医学场景中代价很高。
- 方法: 作者提出 BioChirp，用 LLM 做查询解释与候选过滤，同时结合多源共识实体解析和确定性图检索，在多个数据库中恢复关联关系。
- 结果: 论文报告，与传统 LLM 检索流程相比，BioChirp 在四个主要生物医学数据库上找回了更多关联，并具有更好的可复现性。
- 意义: 这说明在科学检索中，生成模型更适合作为柔性接口，而最终召回与证据整合仍需要确定性结构来兜底。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.25.720782v1

### 6. WindowsWorld: A Process-Centric Benchmark of Autonomous GUI Agents in Professional Cross-Application Environments
- Original Title: WindowsWorld: A Process-Centric Benchmark of Autonomous GUI Agents in Professional Cross-Application Environments
- Keywords: GUI agents, benchmark, cross-application workflows, computer use
- 简要描述: WindowsWorld 用职业场景下的跨应用任务表明，当前 GUI agent 离真实办公流程还很远，尤其在多应用协同和条件判断方面表现脆弱。
- 研究主题: 跨应用 GUI agent 的真实流程评测
- 背景: 现有 GUI agent benchmark 多聚焦单应用、短路径任务，难以反映专业工作流中常见的多软件协同与中间检查。
- 方法: 作者基于 16 类职业场景生成并人工校审任务，在模拟环境中构建包含 181 个任务、17 个桌面应用、平均 5 个子目标的基准，其中 78% 为天然多应用任务。
- 结果: 论文报告，主流大模型和 agent 在多应用任务上的成功率低于 21%，在需要跨 3 个以上应用进行条件判断时尤其容易早期失败，而且执行步数远超人类上限。
- 意义: 这项工作把 GUI agent 的问题暴露得更具体: 瓶颈不只是点击精度，而是跨应用状态整合、流程理解和执行效率。
- 原文链接: https://arxiv.org/abs/2604.27776

### 8. Complete Cyclic Subtask Graphs for Tool-Using LLM Agents: Flexibility, Cost, and Bottlenecks in Multi-Agent Workflows
- Original Title: Complete Cyclic Subtask Graphs for Tool-Using LLM Agents: Flexibility, Cost, and Bottlenecks in Multi-Agent Workflows
- Keywords: tool-using agents, multi-agent workflows, revisitation, coordination cost, routing
- 简要描述: Complete cyclic subtask graph 把多 agent 流程的“最大灵活性”当作实验装置，用来观察回溯能力究竟何时有益、何时只是徒增成本。
- 研究主题: 工具调用 agent 的工作流灵活性与代价分析
- 背景: 长程工具使用任务常需要回头修正或重新探索，但更灵活的多 agent 架构也可能带来大量协调与推理开销。
- 方法: 作者构建完全连通、允许任意 revisitation 的子任务图，并在 TextCraft、ALFWorld 和 Finance-Agent 上对不同 planner、router、tool 暴露方式和轨迹摘要设定进行消融比较。
- 结果: 论文报告，显式回溯在 ALFWorld 这类环境中有助于恢复和探索，但在 TextCraft 这类前置依赖强的任务中往往不如简单前向流程高效；Finance-Agent 的主要瓶颈则更多来自检索、grounding 与证据整合。
- 意义: 这项工作提醒，agent workflow 设计不能只追求灵活，真正重要的是分清失败来自流程结构还是外部信息瓶颈。
- 原文链接: https://arxiv.org/abs/2604.22820

### 9. Reinforced Agent: Inference-Time Feedback for Tool-Calling Agents
- Original Title: Reinforced Agent: Inference-Time Feedback for Tool-Calling Agents
- Keywords: tool-calling agents, inference-time review, reviewer agent, helpfulness-harmfulness
- 简要描述: Reinforced Agent 把工具调用的质量控制前移到执行前，用 reviewer agent 在推理时直接拦截和修正潜在错误。
- 研究主题: 工具调用 agent 的在线审查与纠错
- 背景: 传统 agent 评估大多发生在执行后，发现错误以后通常只能通过重新调 prompt 或重新训练补救，无法在实际运行中即时纠偏。
- 方法: 作者引入专门 reviewer agent，对待执行的工具调用进行事前审查，并提出 helpfulness-harmfulness 指标来量化反馈的净收益与副作用。
- 结果: 论文报告，该方法在 BFCL 和 Tau2-Bench 上带来性能提升，并显示 reviewer 模型选择显著影响收益风险比，自动化 prompt 优化还能进一步增益。
- 意义: 这项工作把 agent reliability 问题从离线评测推进到在线控制，更接近真实生产系统中的守门机制。
- 原文链接: https://arxiv.org/abs/2604.27233

- Original Title: GUI Agents with Reinforcement Learning: Toward Digital Inhabitants

### 11. From Unstructured Recall to Schema-Grounded Memory: Reliable AI Memory via Iterative, Schema-Aware Extraction
- Original Title: From Unstructured Recall to Schema-Grounded Memory: Reliable AI Memory via Iterative, Schema-Aware Extraction
- Keywords: AI memory, schema-grounded records, extraction pipeline, system of record
- 简要描述: 这篇论文的核心观点是，可靠 agent memory 不应只是“能搜回旧文本”，而应像系统记录那样支持事实更新、删除、聚合和显式未知。
- 研究主题: 面向生产级 agent 的结构化外部记忆
- 背景: 很多 memory 系统把历史对话嵌入后做检索，但生产环境需要的是精确事实、当前状态和关系计算，而不是仅靠语义相似度做模糊召回。
- 方法: 作者提出 schema-grounded 的写入路径，将记忆摄取拆为对象识别、字段识别和字段值提取，并用验证门、局部重试和状态化 prompt 控制来保证写入质量。
- 结果: 论文报告，该设计在结构化抽取和端到端 memory benchmark 上都优于对比基线，在需要稳定状态与事实计算的任务上表现尤其明显。
- 意义: 这说明 agent memory 的关键竞争点，不是向量库规模，而是写入时的约束结构与数据契约。
- 原文链接: https://arxiv.org/abs/2604.27906

### 12. In-Context Prompting Obsoletes Agent Orchestration for Procedural Tasks
- Original Title: In-Context Prompting Obsoletes Agent Orchestration for Procedural Tasks
- Keywords: agent orchestration, procedural tasks, in-context prompting, LangGraph
- 简要描述: 这篇论文提出，在流程定义清晰的程序型任务中，把完整流程直接放进 system prompt，可能比外部 orchestrator 更简单也更有效。
- 研究主题: 程序型多轮任务中的自编排替代方案
- 背景: LangGraph、CrewAI 等框架通过外部编排器维护状态和路由，但随着模型能力提升，这种外置结构是否仍然必要值得重新检验。
- 方法: 作者在旅行预订、Zoom 技术支持和保险理赔三类任务中，比较相同模型下的 LangGraph 外部编排与 in-context 自编排方案。
- 结果: 论文报告，in-context 方案在多个质量指标上整体优于外部编排，并在三个任务域中都显著降低了对话失败率。
- 意义: 这项结果并不意味着 orchestration 永远不需要，而是提示流程型任务要先问“模型是否已经足够强”，再决定是否引入额外系统层。
- 原文链接: https://arxiv.org/abs/2604.27891

- 研究主题: 时间序列预测中的角色专用 agent 工作流
- 背景: 许多 LLM-based forecasting 方法仍采用单次从历史到未来的直接映射，难以充分利用多视角证据、迭代修正和集成预测。

- 结果: 论文报告，CastFlow 在多类数据集上取得了优于强基线的总体表现，并显示角色专用设计有助于兼顾通用推理与数值预测能力。
- 意义: 这类工作说明 agent workflow 不只适用于通用任务编排，也正在进入传统预测建模场景，成为组织证据和迭代修正的新范式。
