### 1. Novel Parameter-Free and Interpretable Integration of CITE-seq RNA and ADT Profiles via Tensor Decomposition-Based Unsupervised Feature Extraction
- Original Title: Novel Parameter-Free and Interpretable Integration of CITE-seq RNA and ADT Profiles via Tensor Decomposition-Based Unsupervised Feature Extraction
- Keywords: CITE-seq, tensor decomposition, multi-modal integration, unsupervised learning
- 简要描述: 该工作用张量分解把 CITE-seq 的 RNA 与 ADT 信号做无参数整合，试图在不依赖手工调权的前提下获得更一致的细胞表征。
- 研究主题: CITE-seq 多模态数据的可解释整合
- 背景: RNA 和 ADT 在维度、稀疏性和噪声特征上差异很大，现有整合方法往往依赖预筛基因或模态权重设定。
- 方法: 作者构建 gene × cell × protein 张量并施加 HOSVD，从共享潜变量中同时提取基因、细胞和蛋白表示，并结合稀疏矩阵分块实现大规模计算。
- 结果: 在五个 ImmGen T-cell CITE-seq 数据集上，细胞嵌入整体上比 RNA-only、protein-only 和 totalVI 更贴近标注细胞类型，同时还能进行事后无监督基因选择与质量控制。
- 意义: 这类方法有价值的地方，在于它把多模态整合从“调参工程”转成更可解释的结构分解问题。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.18.719420v1

### 2. Foundation cell segmentation models performance on live microscopy and spatial-omics data
- Original Title: Foundation cell segmentation models performance on live microscopy and spatial-omics data
- Keywords: cell segmentation, microscopy, spatial omics, benchmarking
- 简要描述: 论文系统比较多类通用细胞分割模型在活细胞显微图像和空间组学数据上的表现，并关注其对下游分析的影响。
- 研究主题: 细胞分割基础模型的跨场景评测
- 背景: 通用分割模型数量快速增长，但它们在不同成像模态和下游生物分析中的真实差异还缺少系统比较。
- 方法: 作者评测了 Cellpose、Cellpose-SAM、uSAM、CellSAM、Mesmer、InstanSeg 等模型，覆盖 phase contrast、荧光细胞培养和 CODEX 组织成像场景。
- 结果: Cellpose-SAM 在相差图像上表现较强，SAM 类模型在荧光细胞培养数据上较稳，但在 CODEX 数据上不存在单一通吃模型，不同模型会显著影响聚类和细胞类型识别。
- 意义: 这提醒空间组学工作流中的 segmentation 不是前置琐事，而是会直接改变后续生物结论的关键步骤。
- 原文链接: https://www.biorxiv.org/content/10.64898/2026.04.18.719315v1

- 背景: 传统 signature 分析擅长描述产生了哪类碱基替换，但对这些突变偏好出现在基因组哪里刻画不足。
